# Weather

一款简单的天气软件.

## Introduction
1. 天气信息来源于和风天气;
2. 支持分享天气信息到微信朋友圈;
3. 可查看最近七天的天气情况,部分城市还可以查看 PM2.5 的值;
4. 支持手动更新检测.
